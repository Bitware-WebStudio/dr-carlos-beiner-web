# Content Placeholders

Please replace the following placeholders with real information.

## Global
- `{{WHATSAPP_NUMERO}}`: Phone number for WhatsApp (e.g., +56912345678)
- `{{BOOKING_URL}}`: URL for the scheduling system (e.g., Dentalink link)

## Hero Section
- `{{HERO_TITULO}}`: Main headline.
- `{{HERO_SUBTITULO}}`: Subheadline calling for calm/professionalism.

## Services
- `{{SERVICIO_1_TITULO}}`: Title of Service 1.
- `{{SERVICIO_1_DESC}}`: Description of Service 1.
- `{{SERVICIO_2_TITULO}}`: Title of Service 2.
- `{{SERVICIO_2_DESC}}`: Description of Service 2.
... (Add more as needed)

## Reviews
- `{{GOOGLE_REVIEWS_EMBED}}`: HTML/Script for Google Reviews widget.
- `{{RATING_SCORE}}`: Current rating (e.g., 4.8).

## Location
- `{{METRO_DETAILS}}`: Information about nearest Metro station.
- `{{PARKING_DETAILS}}`: Information about parking availability.
- `{{MAPS_URL}}`: Direct link to Google Maps location.

## About Doctor
- `{{BIO_EXPERIENCE}}`: Bullet point about experience.
- `{{BIO_HUMAN}}`: Bullet point about human approach.
- `{{BIO_CLARITY}}`: Bullet point about clarity of explanation.

## First Visit
- `{{PRIMERA_VISITA_PASO_1}}`: Text for Step 1 (Contact).
- `{{PRIMERA_VISITA_PASO_2}}`: Text for Step 2 (Evaluation).
- `{{PRIMERA_VISITA_PASO_3}}`: Text for Step 3 (Treatment Plan).
- `{{PRIMERA_VISITA_PASO_4}}`: Text for Step 4 (Payment/Options).
- `{{FINANCING_ICONS}}`: Logos or text for payment methods (e.g., "12 Cuotas").

## Media
- `{{HERO_VIDEO_URL}}`: URL for the background video of the hero section.
- `{{TESTIMONIAL_VIDEO_1}}`: URL for patient testimonial video 1.
- `{{TESTIMONIAL_VIDEO_2}}`: URL for patient testimonial video 2.
- `{{TESTIMONIAL_VIDEO_3}}`: URL for patient testimonial video 3.
